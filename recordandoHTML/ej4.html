• URL: dirección única y específica de un recurso en Internet, como un sitio web, una página, una imagen o un archivo
  • HTTP: protocolo fundamental de la World Wide Web que define las reglas para la comunicación entre un navegador (cliente) y un servidor web
  • TCP/IP: conjunto de protocolos de red que definen cómo se comunican los dispositivos en redes, siendo la base de Internet.
  • Internet: Red informática mundial, descentralizada, formada por la conexión directa entre computadoras mediante un protocolo especial de comunicación.
  • W3C: comunidad internacional dedicada a desarrollar estándares para la web, como HTML y CSS,
  • HTML: Lenguaje de Marcado de Hipertexto
